# Личный сайт
Адаптивный личный сайт с использованием HTML5, CSS3 и JavaScript

## Функционал
- 5 страниц: Главная, Обо мне, Навыки, Работы, Контакты
- Переключение темной и светлой темы
- Валидация формы обратной связи
- Адаптивный дизайн для мобильных и десктопов

## Технологии
HTML5, CSS3, JavaScript, Flexbox, медиазапросы
